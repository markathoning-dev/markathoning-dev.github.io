    Website Purpose & Identity

    This website documents TAQ Cleaner - a utility for cleaning and classifying crypto exchange TAQ (Trade and Quote) data.

    What We Stand For

    • Data Integrity: Providing reliable tools for financial data processing
    • Performance: Optimized for speed and efficiency in handling large datasets
    • Transparency: Open documentation that's easy to understand and follow
    • Developer Experience: First-class CLI and API interfaces

    Website Goals

    1. Educate: Help users understand how to use TAQ Cleaner effectively
    2. Enable: Provide clear examples and documentation for integration
    3. Showcase: Demonstrate the power and flexibility of the tool
    4. Support: Offer comprehensive reference material for troubleshooting
